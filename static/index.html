API 키 설정
아래 3개 키를 모두 입력해야 합니다. 처음이라면 발급 방법 보기를 클릭하세요.
① Claude AI 필수
AI 신호 분석 엔진에 사용됩니다.  console.anthropic.com →
  1. console.anthropic.com 접속 후 회원가입 / 로그인
  2. 상단 메뉴 API Keys 클릭
  3. Create Key 버튼 → 이름 입력 후 생성
  4. 표시된 키(sk-ant-...)를 복사해 위에 붙여넣기
⚠ 키는 생성 직후에만 표시됩니다. 반드시 바로 복사하세요.
② Binance 필수
선물 시장 데이터 · 계좌 조회에 사용됩니다.  binance.com →
  1. 바이낸스 API 관리 접속
  2. Create API 클릭 → System generated 선택
  3. API 이름 입력 (예: btc-analyzer) 후 2FA 인증
  4. 생성된 API Key와 Secret Key를 각각 복사
  5. Edit restrictions 클릭 → 권한 설정:
    • ✓ 필요 Enable Reading
    • ✓ 필요 Enable Futures
    • ✗ 제외 Enable Withdrawals — 절대 체크 금지
    • ✗ 제외 Enable Spot & Margin Trading
⚠ Secret Key는 생성 시 한 번만 표시됩니다. 즉시 복사하세요.
출금 권한을 부여하면 계좌 자산이 위험에 노출됩니다.
